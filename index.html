Frank's Cheat Sheet

Semantic HTML - Section Tags

Tag Name Description
<header> Header The header of a content section or the web page. The web page header often contains the website branding or logo.
<nav> Nav The navigation links of a section or the web page.
<footer> Footer The footer of a content section or the web page. On a web page, it often contains secondary links, the copyright notice, privacy policy and -cookie policy links.
<main> Main Specifies the main content of a section or the web page.
<article> Article An independent, self-contained block of content such as a blog post or product.
<section> Section A standalone section of the document that is often used within the body and article elements.
<details> Details A collapsed section of content that can be expanded if the user wishes to view it.
<summary> Summary Specifies the summary or caption of a <details> element.
<h1><h2><h3><h4><h5><h6> Headers Headings on the web page. <h1> indicates the most important heading whereas <h6> indicates the least important.

Semantic HTML - Content Tags

Tag Name Description
<blockquote> Quotes Used to describe a quotation.
<dl> Description List Used to define a description list.
<dt> Descripton Terms Used to describe terms inside <dl> elements.
<dd> Description Definition Used to define a description for the preceding <dt> element.
<figcaption> Figure Caption Defines a caption for a photo image.
<figure> Figure Applies markup to a photo image.
<hr> Horizontal Line Adds a horizontal line to the parent element.
<li> List Item Used to define an item within a list.
<menu> Menu A semantic alternative to <ul> tag.
<ol> Ordered List Defines an ordered list.
<p> Paragraph Defines a paragraph.
<pre> Preformatted Used to represent preformatted text. Typically rendered in the web browser using a monospace font.
<ul> Unordered List Unordered list

Semantic HTML - Inline Tags

Tag Name Description
<a> Anchor An anchor link to another HTML document.
<abbr> Abbreviation Specifies that the containing text is an abbreviation or acronym.
<b> Bold Bolds the containing text. When used to indicate importance use <strong> instead.
<br> Line Break A line break. Moves the subsequent text to a new line.
<cite> Title For Creative Work Defines the title of creative work (for example a book, poem, song, movie, painting or sculpture). The text in the <cite> element is usually rendered in italics.
<code> Code Block Indicates that the containing text is a block of computer code.
<data> Machine Data Indicates machine-readable data.
<em> Emphasize Text Emphasizes the containing text.
<i> Italics The containing text is displayed in italics. Used to indicate idiomatic text or technical terms.
<mark> Highlighted The containing text should be marked or highlighted.
<q> Short Quote The containing text is a short quotation.
<s> Strike Through Displays the containing text with a strikethrough or line through it.
<samp> Sample The containing text represents a sample.
<small> Legal Text Used to represent small text, such as copyright and legal text.
<span> Span A generic element for grouping content for CSS styling.
<strong> Important Text Displays the containing text in bold. Used to indicate importance.
<sub> Subscript The containing text is subscript text, displayed with a lowered baseline.
<sup> Superscript The containing text is superscript text, displayed with a raised baseline.
<time> Date and Time A semantic tag used to display both dates and times.
<u> Underline Displays the containing text with a solid underline.
<var> Variable The containing text is a variable in a mathematical expression.

Semantic HTML - Embeded and Media Tags

Tag Name Description
<audio> Audio Used to embed audio in web pages.
<canvas> Canvas Used to render 2D and 3D graphics on web pages.
<embed> Embed Used as a containing element for external content provided by an external application such as a media player or plug-in application.
<iframe> iFrame Used to embed a nested web page.
<img> Image Embeds an image on a web page.
<object> Object Similar to <embed> but the content is provided by a web browser plug-in.
<picture> Image An element that contains one <img> element and one or more <source> elements to offer alternative images for different displays/devices.
<video> Video Embeds a video on a web page.
<source> Media Resource Specifies media resources for <picture>, <audio> and <video> elements.
<svg> SVGs Used to define Scalable Vector Graphics within a web page.

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<caption> Caption Defines the caption of a table element.
<colgroup> Column Group Defines a semantic group of one or more columns in a table for formatting.
<col> Semantic Column Defines a semantic column in a table.
<tfoot> Table Foot The set of rows defining the footer in a table.

Table Tags

Selector Name Description
*{property:value;} Universal Selects all elements on the pages.
p {property:value;} Type Selects all HTML tag/element of given type in your document.
#id {property:value;} ID Selects an element based on the value of its unique id attribute.
.class {property:value;} Class Selects all elements in the document that have the given class attribute.
a[attribute=value] {property:value;} Attribute Selects all elements that have a specified attribute.
selector1 selector2/ selector 1+selector2 / selector 1> selector 2 {property: value;} Combinators Complex selectors consisting of more than one selectors having some relationship between them.
selector: pseudo-class{property: value; } Pseudo Define the special state of an element to add an effect to an existing element based on its states.
